Submission

Provide a Google Form that has a single email field and a I'm not a robot reCAPTCHA. The server opens the form in a headless browser, solves the captcha via the audio challenge, and submits each email.

Google sign-in

One-time sign-in: opens Chrome on the server pointed at Google's sign-in page, you log in manually, the cookies are saved. Every subsequent form submission clones that signed-in profile so the form sees the user as logged in. Sign-in is intentionally NOT proxied — your real IP avoids "unusual sign-in activity" prompts.
