Google Form Submitter

Single-email forms with audio reCAPTCHA bypass

Submission

Provide a Google Form that has a single email field and a I'm not a robot reCAPTCHA. The server opens the form in a headless browser, solves the captcha via the audio challenge, and submits each email.

After Google accepts each form, log in to a Gmail / IMAP inbox and wait for the receipt email from forms-receipts-noreply@google.com. If the receipt never arrives within the timeout, the submission is marked failed. Useful to catch silent drops where the response counter goes up but no email actually lands.
Gmail: requires 2-Step Verification + a 16-character App Password (regular passwords are rejected).

Recommended for avoiding Google's "Try again later" rate limit. Pre-filled with DataImpulse gateway credentials. Each browser restart opens a new TCP session, which forces a new exit IP. Combine with the retry count on the right for automatic IP rotation when Google blocks a session.
